🍜 한국 전통 요리 탐험

Korean Traditional Food Explorer

🎯 프로젝트 목적

이 웹앱은 한국의 아름다운 전통 요리 문화를 지역별로 탐험할 수 있도록 만들어졌습니다. 각 지역의 고유한 음식 문화와 그 뒤에 숨겨진 역사적, 지리적 배경을 쉽고 재미있게 학습할 수 있습니다.

🗺️ 활용 방법

  • 지도 탐험: 지도에서 관심 있는 도시 마커를 클릭하여 해당 지역의 전통 요리를 확인하세요
  • 음식 검색: 상단 검색창에 음식 이름을 입력하면 해당 음식이 있는 지역이 하이라이트됩니다
  • 상세 정보: 각 요리를 클릭하면 유래, 역사적 배경, 지리적 특징을 자세히 볼 수 있습니다
  • 다국어 지원: 우측 상단의 언어 버튼으로 한국어/영어를 전환할 수 있습니다

👨‍💻 제작자 정보

이 프로젝트는 한국의 전통 음식 문화를 널리 알리고자 하는 마음으로 개발되었습니다. 순수 HTML, CSS, JavaScript로 구현되어 GitHub Pages에서 쉽게 배포 가능하도록 설계되었습니다.

개발 목표: 사용자 친화적인 인터페이스를 통해 한국 전통 요리의 아름다움과 깊이를 전 세계에 소개하는 것

🛠️ 사용된 오픈소스 기술

🗺️ Leaflet.js

인터랙티브 지도 라이브러리로 사용자 친화적인 지도 인터페이스를 제공합니다.

🌍 OpenStreetMap

오픈소스 지도 타일 서비스로 무료이면서도 상세한 지도 데이터를 제공합니다.

💻 Vanilla JavaScript

별도의 프레임워크 없이 순수 JavaScript로 구현하여 빠른 로딩과 가벼운 용량을 실현했습니다.

🎨 CSS3

현대적인 CSS 기능들을 활용하여 아름답고 반응형인 디자인을 구현했습니다.

📊 기술적 특징

  • 반응형 디자인: 모바일, 태블릿, 데스크톱 모든 기기에서 최적화된 경험
  • 다국어 지원: 한국어/영어 실시간 전환 기능
  • 검색 기능: 실시간 음식 검색 및 자동 하이라이트
  • 모듈화된 데이터: JSON 파일을 통한 확장 가능한 도시/음식 데이터 구조
  • 성능 최적화: 빠른 로딩과 부드러운 애니메이션
  • 접근성: 시맨틱 HTML과 키보드 네비게이션 지원

🚀 향후 계획

이 프로젝트는 지속적으로 발전해 나갈 예정입니다:

  • 더 많은 도시와 전통 요리 추가
  • 음식 레시피 및 영양 정보 제공
  • 사용자 리뷰 및 평점 시스템
  • 음식점 위치 정보 연동
  • 소셜 미디어 공유 기능
